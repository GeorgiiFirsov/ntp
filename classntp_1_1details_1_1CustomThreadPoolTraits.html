Detailed Description

Traits for a custom threadpool.

These traits create a new threadpool in constructor and then allow user to interact with this pool via ThreadPool class.

DWORD  max_threads = 0 
)

Constructor with the ability to set threadpool threads number.

If min_threads is 0, then minimum number of threads will be set to 1. If max_threads is 0 or less than min_threads, then maximum number of threads will be set to ntp::details::HardwareThreads. If max_threads is still less than min_threads after changes described above, then maximum number of threads is set to be equal to minimum.
